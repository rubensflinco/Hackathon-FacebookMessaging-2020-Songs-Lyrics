Bem vindo(a) ao projeto Qual a letra dessa música?

Nosso projeto foi desenvolvido para facilitar o dia-a-dia das pessoas que amam música, os amantes de letras musicais que estão lendo agora não precisam se esforçar muito para poderem ler suas letras favoritas.

É super simples de usar o nosso aplicativo, você só tem uma conta no Facebook e usa o aplicativo Messeger ou até o site do Facebook.

Com apenas um desses aplicativos ou site, você já pode conversar com o nosso bot, que permite pesquisar letras instantaneamente.

Vídeo

Como usar?

Então você curtiu mesmo nosso projeto em!

Bom para usar é bem simples, basta acessar na pagina de Facebook e clicar no botão "Enviar Mensagem".

Logo após fazer o você vai começar a conversar com o bot dai pra frente é só seguir o passo a passo, ou digitar o nome da música, artista ou albun que quer saber a letra.

Inspiração

Nossa inspiração foi ver aplicativos como "Shazam" e "Letras.com" que são pesados e pensamos por que não ter a mesma função que eles usando o Facebook Messenger, no futuro pretendemos tornar as funções mais semelhantes à forma de pesquisa do aplicação "Shazam".

O que faz

Nosso projeto, Songs Lyrics, é um aplicativo integrado ao facebook messenger para facilitar a vida dos amantes da música em todo o mundo, tornando as músicas e os artistas muito mais acessíveis, basta enviar uma mensagem para a nossa página do facebook e nosso bot te ajudará a encontrar o que você está procurando, simples assim!

Como a construímos

Usamos o PHP para criar a API de pesquisa de letras de músicas, as APIs foram construídas por nós mesmos, mas hospedadas em servidores do site parceiro do nosso amigo, que já possuía o site com um grande banco de letras de músicas e ele nos deu autorização para usar para criar isso projeto livre.

Já escrevemos as mensagens usando a plataforma ChatFuel conectada à página do facebook e, nessa plataforma, escolhemos o plano gratuito, usamos porque foi uma das únicas que nos permitiu solicitar uma API para obter os valores dinamicamente e responda ao usuário as letras musicais.

Desafios que encontramos

A primeira grande dificuldade foi quando pensamos em "como vamos conseguir tantas letras musicais?". E então um membro da equipe disse que conhecia um amigo dono de um site de letras de músicas e nos deu permissão para usar o banco de dados em o site dele em nosso aplicativo.

A segunda dificuldade foi encontrar uma plataforma para escrever as mensagens que o bot havia enviado, a maioria das plataformas não tinha a opção de enviar solicitações de APIs no plano gratuito, mas depois de alguns dias de pesquisa, encontramos o ChatFuel de que participamos muito bem. a criação deste projeto.

Realizações das quais nos orgulhamos

Estamos orgulhosos de ser o primeiro a criar um bot automático no facebook messenger para procurar letras musicais.

O que aprendemos

Tivemos que aprender a editar o vídeo para tornar nossa apresentação de bot mais profissional. E aprendemos tudo sobre a plataforma ChatFuel e sua documentação de conexão API.

O que vem a seguir para músicas Lyrics

Queremos procurar maneiras de tornar a vida de nossos usuários ainda mais fácil, por isso investiremos nosso tempo em pesquisas para melhorar a maneira como nossos usuários pesquisam suas letras musicais, possibilitando pesquisas por áudio, imagem ou texto. E é claro que queremos mais do que tudo ter a opção de visualizar as palavras junto com a escrita das letras.
